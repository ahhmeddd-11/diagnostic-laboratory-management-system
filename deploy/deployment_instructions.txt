UNILAB DIAGNOSTIC SYSTEM - DEPLOYMENT INSTRUCTIONS
==================================================

Welcome! These instructions will guide you through installing the Unilab Diagnostic Management System on the main computer in your laboratory. Once installed, it will run automatically and can be accessed from any other computer or tablet in your clinic.

REQUIREMENTS:
-------------
- The computer hosting this software MUST have MySQL Server installed and running.
- The default database name must be `diagnostic_lab`.
- If your MySQL credentials are not 'root' and 'Ahmed', please update them in the `.env` file BEFORE building.


STEP 1: DATABASE SETUP (MANUAL)
-------------------------------
Before installing the software, you must create the database structure:
1. Open MySQL Command Line or MySQL Workbench on the target computer.
2. Open the `database\schema.sql` file provided in this project.
3. Execute the entire `schema.sql` file in one shot.
   This will automatically:
   - Create the `diagnostic_lab` database.
   - Build all necessary tables.
   - Insert the default system administrator (Email: admin@lab.com, Password: admin123).


STEP 2: PREPARE THE FILES
-------------------------
1. Open the `deploy\` folder and double-click the `build.bat` file.
2. A black terminal window will appear. It will automatically compile the server and backup systems into executables.
3. Wait for it to say "BUILD COMPLETE!" and generate files in the `dist\` folder at the root.


STEP 3: CREATE THE WINDOWS INSTALLER (Setup.exe)
------------------------------------------------
This step turns your files into a professional Windows Installation Wizard.
1. Download and install **Inno Setup** (it is a free, safe tool) from:
   https://jrsoftware.org/isdl.php
2. Once installed, right-click the `unilab_installer.iss` file in this folder and select "Open with Inno Setup".
3. In the Inno Setup window, click the green "Play/Compile" button at the top.
4. It will generate a new folder called `installer\` containing `Unilab_Setup_v1.0.exe`.


STEP 4: INSTALL THE SYSTEM
--------------------------
1. Copy `Unilab_Setup_v1.0.exe` to the target computer in the lab.
2. Double-click it to run the installer.
3. Follow the standard "Next -> Next -> Install" wizard.
4. The installer will AUTOMATICALLY:
   - Copy all files to `C:\Program Files\Unilab`
   - Create Desktop and Start Menu shortcuts
   - Set up the server to launch automatically on Windows boot
   - Register the daily database backup (runs silently every night at 11:00 PM)


STEP 5: STARTING THE SYSTEM
---------------------------
1. You can start the system at any time by double-clicking the "Unilab Diagnostic Server" shortcut on your Desktop.
2. A small console window will open. DO NOT CLOSE IT! If you close this window, the system turns off.
3. After 2 seconds, your default web browser will automatically open to the login screen.


HOW TO ACCESS FROM OTHER COMPUTERS (LAN SUPPORT)
------------------------------------------------
This system is designed so doctors, technicians, and receptionists can all use it simultaneously from different devices.

1. On the Main Server Computer (where the .exe is running):
   - Open Command Prompt (search 'cmd' in Windows Start Menu).
   - Type `ipconfig` and press Enter.
   - Look for the "IPv4 Address" (it usually looks like 192.168.1.XX).

2. On any other computer/tablet connected to the same Wi-Fi/Network:
   - Open a web browser (Chrome/Edge).
   - Type the IPv4 address followed by :5000 in the URL bar.
     Example: http://192.168.1.55:5000
   - You will see the Unilab login screen!


Enjoy your new locally hosted, secure, and automated Diagnostic Management System!
