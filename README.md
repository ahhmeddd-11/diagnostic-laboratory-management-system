# 🧪 UNILAB Diagnostic Laboratory Management System

A production-ready **Diagnostic Laboratory Management System** developed using **Python, Django, MySQL, HTML, CSS, JavaScript, and PyWebView**. The application is designed for diagnostic laboratories to efficiently manage patients, laboratory tests, billing, and professional report generation through a native Windows desktop application.

---

## 📖 Overview

UNILAB Diagnostic Laboratory Management System streamlines the complete laboratory workflow—from patient registration to report printing.

The system provides a modern desktop interface while leveraging Django as the backend framework and MySQL as the database. The application is packaged as a standalone Windows executable with a professional installer, eliminating the need for Python installation on client machines.
